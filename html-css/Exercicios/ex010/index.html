Links internos

Para acessar uma página interna na mesma pasta dentro do servidor, você só precisa do nome do arquivo no link do parâmetro "a".

Para acessar uma página interna, porém em outra pasta no mesmo servidor, você deve referenciar o caminho completo ao arquivo dessa página no link do parâmetro "a".
